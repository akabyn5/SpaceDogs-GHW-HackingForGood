## Welcome to GitHub!

People use GitHub to build some of the most advanced technologies in the world. Whether you’re visualizing data or building a new game, there’s a whole community and set of tools on GitHub that can help you do it even better.

**GitHub Skills’ Introduction to GitHub** exercise guides you through everything you need to start contributing in less than an hour.

---

## What You'll Learn

- How to navigate GitHub repositories
- Creating and editing files
- Making your first pull request
- Collaborating with the global developer community

![Processing](https://github.com/akabyn5/SpaceDogs-GHW-HackingForGood/blob/main/docs/old-submissions/Introduction%20of%20Github/IMG-20260313-WA0145.jpg?raw=true)
## Getting Started

This repository is part of the official GitHub Skills course. By completing the exercises here, you will:

1. Fork this repository
2. Edit files directly in the browser
3. Submit a pull request with your changes
4. Merge your contribution
