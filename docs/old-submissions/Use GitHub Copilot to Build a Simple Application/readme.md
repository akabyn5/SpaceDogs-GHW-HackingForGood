# Choose Your Own Adventure Game

Built with GitHub Copilot — A simple command-line interactive story game.

This project demonstrates how GitHub Copilot can quickly generate a complete, functional application from natural language prompts. The entire game logic, story branches, and user interaction were created with the help of Copilot.

## Overview

GitHub Copilot prompts are incredibly effective for generating code for specific use cases. In this example, Copilot helped build a complete "Choose Your Own Adventure" game in Python. 

The game features multiple story paths, player choices, and a simple win/lose condition — all generated iteratively through conversation with Copilot.

## Features

- Interactive command-line interface
- Branching story with multiple endings
- Simple state management
- Easy to extend with new scenes

## How It Was Built

1. Started with a prompt: *"Create a Python command line choose your own adventure game with at least 3 different endings."*
2. Iteratively refined scenes and logic using follow-up prompts.
3. Added input validation and error handling with Copilot's suggestions.
4. Polished the narrative and user experience.

This project shows how quickly you can go from idea to working application using GitHub Copilot.
