# 🚀 Space Dogs Community Hub

## Project Overview

### Project Purpose

Space Dogs Community Hub is an open-source web platform designed to support nonprofit organizations, educational institutions, STEM communities, and volunteer initiatives through technology.

The project was initiated during Global Hack Week: Hacking for Good as a practical exercise in collaborative software engineering, GitHub workflows, and AI-assisted development. Beyond the hackathon, the platform is intended to become a reusable foundation for future Space Dogs outreach and educational projects.

The application aims to centralize volunteer management, educational resources, community events, and collaborative opportunities while serving as a learning environment for modern software development practices.

---

## Objectives

### Primary Objectives

- Develop a collaborative full-stack web application following industry best practices.
- Strengthen the International Projects team's experience with GitHub, GitHub Copilot, Codespaces, and collaborative software development.
- Build reusable software architecture for future Space Dogs projects.
- Create a platform that can support nonprofit organizations and STEM education initiatives.
- Improve frontend and backend development skills through practical implementation.

### Technical Objectives

- Apply GitHub Flow and collaborative development workflows.
- Learn and implement GitHub Copilot Plan Mode and Agent Mode.
- Design a scalable software architecture.
- Produce clear technical documentation.
- Practice clean repository organization and version control.
